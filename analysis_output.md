## 4. Research Gap (Identifikasi Lengkap)

### Gap 1: AI/Teknologi Belum Diuji sebagai Moderator
**Deskripsi**: Tidak ada tesis S2 sektor teknologi BEI yang menguji apakah adopsi AI (diukur melalui disclosure index, intangible intensity, atau digital capex) memperkuat/memperlemah hubungan rasio fundamental → outcome perusahaan.
**Justifikasi**: Literatur internasional 2024–2025 mulai menunjukkan GenAI berdampak pada sentimen pasar dan valuasi (ScienceDirect, Springer).
**Urgensi**: Tinggi — sektor teknologi adalah yang paling terdampak oleh AI.

### Gap 2: Structural Break Era GenAI (2022/2023) Belum Diuji
**Deskripsi**: Peluncuran ChatGPT (Nov 2022) dan adopsi massal GenAI di 2023 menciptakan potensi structural break. Belum ada tesis yang menguji apakah relevansi rasio fundamental berubah sebelum vs sesudah era GenAI.
**Justifikasi**: Event study dan DID framework cocok untuk menguji regime change ini.
**Urgensi**: Tinggi — timing yang tepat untuk penelitian ini (data pre & post sudah tersedia).

### Gap 3: Metode Kausal Masih Jarang
**Deskripsi**: Mayoritas tesis menggunakan OLS/panel standar tanpa menangani endogeneity. GMM, DID, instrumental variable, dan event study masih jarang.
**Justifikasi**: Tanpa metode kausal, hubungan yang ditemukan bisa bersifat korelasi saja.
**Urgensi**: Sedang — memperkuat kontribusi metodologis tesis.

### Gap 4: Belum Ada Multi-Dependen dalam Satu Studi
**Deskripsi**: Tesis yang ada hanya menguji 1 dependen (harga ATAU return ATAU firm value). Belum ada yang menguji ketiganya secara simultan untuk melihat konsistensi pengaruh rasio fundamental.
**Justifikasi**: Rasio yang signifikan terhadap harga belum tentu signifikan terhadap return atau Tobin's Q.
**Urgensi**: Sedang — menambah kedalaman analisis.

### Gap 5: Periode 2019–2025 Belum Tercakup Penuh
**Deskripsi**: Tesis yang ada hanya mencakup sub-periode (2020–2023 atau 2021–2023). Periode 2019–2025 yang mencakup pre-COVID, COVID, recovery, dan era GenAI belum diuji.
**Justifikasi**: Panel yang lebih panjang memungkinkan analisis regime dan structural break yang lebih robust.
**Urgensi**: Sedang — menambah cakupan temporal.

### 5.2 Pemilihan Variabel

| Kategori | Rekomendasi | Alasan |
|----------|-------------|--------|
| **Dependen** | Harga saham (closing price), Return saham (annual), Nilai perusahaan (Tobin's Q) | Tobin's Q lebih baik dari PBV karena memasukkan liabilitas |

### 5.3 Metodologi

| Aspek | Rekomendasi |
|-------|-------------|
| **Model utama** | Panel data regression (Fixed Effect / Random Effect) dengan robust standard errors |
| **Pemilihan FE/RE** | Hausman test |
| **Moderasi** | Interaction term: Rasio x AI_Disclosure, Rasio x GenAI_Dummy |
| **Endogeneity** | System GMM sebagai robustness check |
| **Structural break** | Chow test atau dummy interaction |
| **Diagnostik** | VIF (multikolinearitas), Breusch-Pagan (heteroskedastisitas), Wooldridge (autokorelasi) |
| **Robustness** | Subsample pre/post 2022, winsorizing 1%/5%, alternatif proksi (PBV vs Tobin's Q) |
| **Software** | Stata / EViews / R (plm package) |

### 5.4 Konstruksi AI Disclosure Index

**Metode yang direkomendasikan**: Content analysis manual atau semi-otomatis pada annual report.

**Langkah**:
1. Download annual report perusahaan teknologi BEI 2019–2025 dari IDX
2. Buat daftar kata kunci AI: "artificial intelligence", "machine learning", "deep learning", "kecerdasan buatan", "big data", "automation", "digital transformation", "AI", "IoT", "cloud computing"
3. Hitung frekuensi kemunculan (word count) atau buat skor biner (ada/tidak ada) per kategori
4. Normalisasi menjadi indeks 0–1
5. Validasi: inter-coder reliability jika manual, atau gunakan text mining (Python/R) jika semi-otomatis

| Aspek | Detail |
|-------|--------|
| **Tujuan** | Membandingkan rasio fundamental perusahaan teknologi Indonesia vs AS |
| **Sumber data AS** | SEC EDGAR via `data.sec.gov` API (sudah disiapkan di `scripts/sec_fetch.ps1`) |

### 5.6 Peringatan & Hal yang Harus Dihindari

| Jangan | Alasan |
|--------|--------|
| Jangan hanya pakai OLS tanpa panel | Data panel memerlukan FE/RE; OLS pooled mengabaikan heterogeneity |
| Jangan gunakan terlalu banyak variabel independen | Dengan sampel kecil (sektor teknologi BEI ~30–40 perusahaan), terlalu banyak variabel menyebabkan overfitting |
| Jangan abaikan multikolinearitas | Rasio profitabilitas (ROA, ROE, NPM) cenderung berkorelasi tinggi — pilih atau gunakan PCA |
| Jangan klaim kausalitas dari regresi biasa | Gunakan bahasa "pengaruh" atau "asosiasi", bukan "menyebabkan", kecuali pakai metode kausal |
| Jangan abaikan outlier | Sektor teknologi cenderung volatile — winsorize atau trimming wajib |

## 8. Metodologi Appendix

### A. Definisi Variabel dan Rumus

**Variabel Dependen:**
| Variabel | Rumus | Sumber Data |
|----------|-------|-------------|
| Harga Saham (PRICE) | Closing price akhir tahun | IDX / Yahoo Finance |
| Return Saham (RET) | (P_t - P_{t-1}) / P_{t-1} | Dihitung dari closing price |
| Tobin's Q (TQ) | (Market Cap + Total Liabilities) / Total Assets | Laporan keuangan + harga saham |

**Variabel Independen (Rasio Fundamental):**
| Variabel | Rumus |
|----------|-------|
| ROA | Net Income / Total Assets |
| ROE | Net Income / Total Equity |
| NPM | Net Income / Revenue |
| CR | Current Assets / Current Liabilities |
| DER | Total Liabilities / Total Equity |
| TATO | Revenue / Total Assets |
| EPS | Net Income / Shares Outstanding |

**Variabel Moderasi:**
| Variabel | Rumus / Definisi |
|----------|-----------------|
| AI Disclosure Index (AID) | Skor 0–1 dari content analysis annual report (kata kunci AI/digital) |
| Intangible Intensity (II) | Intangible Assets / Total Assets |
| GenAI Dummy (DGENAI) | 1 jika tahun >= 2023, 0 sebaliknya |

**Variabel Kontrol:**
| Variabel | Rumus |
|----------|-------|
| SIZE | Ln(Total Assets) |
| GROWTH | (Revenue_t - Revenue_{t-1}) / Revenue_{t-1} |
| AGE | Tahun observasi - Tahun listing |
| VOL | Standard deviation of monthly returns |

### B. Skema Dataset

```
firm_id | ticker | year | sector | price | return | tobins_q
roa | roe | npm | cr | der | tato | eps
size | growth | age | volatility
ai_disclosure_index | intangible_intensity | genai_dummy
```

### C. Model Empiris

**Model 1 — Baseline:**
```
Y_it = α + β₁ROA_it + β₂ROE_it + β₃NPM_it + β₄CR_it + β₅DER_it
       + β₆TATO_it + β₇EPS_it + γControls_it + μ_i + ε_it
```
Di mana Y = PRICE, RET, atau TQ (diestimasi terpisah untuk masing-masing)

**Model 2 — Moderasi AI Disclosure:**
```
Y_it = α + β₁X_it + β₂AID_it + β₃(X_it × AID_it)
       + γControls_it + μ_i + ε_it
```

**Model 3 — Structural Break GenAI:**
```
Y_it = α + β₁X_it + β₂DGENAI_t + β₃(X_it × DGENAI_t)
       + γControls_it + μ_i + ε_it
```

### D. Uji Diagnostik
- **Multikolinearitas**: VIF < 10
- **Heteroskedastisitas**: White test / Breusch-Pagan test
- **Autokorelasi**: Wooldridge test (panel)
- **Model selection**: Hausman test (FE vs RE), Chow test (poolability)
- **Stationarity** (jika panel panjang): Levin-Lin-Chu / Im-Pesaran-Shin

### E. Robustness Checks
- System GMM (lagged dependent variable sebagai instrument)
- Subsample: pre-GenAI (2019–2022) vs post-GenAI (2023–2025)
- Winsorizing outlier pada persentil 1% dan 99%
- Alternatif proksi: PBV sebagai pengganti Tobin's Q
- Exclude COVID year (2020) sebagai sensitivity test
