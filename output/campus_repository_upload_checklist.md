# Campus Repository Upload Checklist

## Berkas Final
- [ ] Tesis final `.pdf` sesuai format kampus
- [ ] Tesis final `.docx` (jika diminta)
- [ ] Ringkasan/abstrak bilingual
- [ ] Surat pernyataan/orisinalitas bertanda tangan
- [ ] Lembar pengesahan (jika diwajibkan)

## Metadata Repositori
- [ ] Judul final sudah konsisten
- [ ] Nama penulis dan NIM benar
- [ ] Kata kunci sudah final
- [ ] Tahun publikasi benar
- [ ] Pembimbing/penguji tercantum sesuai dokumen resmi

## Bukti dan Arsip
- [ ] Bukti unggah (screenshot/receipt) tersimpan
- [ ] Tautan repositori kampus dicatat
- [ ] Salinan berkas final diarsipkan lokal
