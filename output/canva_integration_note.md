# Canva Integration Note

## Status
Direct Canva account/API connection cannot be executed from this environment (no interactive OAuth/login support).

## Working Alternative (Implemented)
Canva supports importing PowerPoint files. A Canva-ready deck has been generated:
- `draft/defense_slides.pptx`

## Upload Steps in Canva
1. Open Canva.
2. Create a new Presentation.
3. Use **Import file** and upload `draft/defense_slides.pptx`.
4. Apply your Canva brand/theme and export final slides if needed.
